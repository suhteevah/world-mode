## Summary

### Key Lessons
- Dashboard monitoring is essential for debugging a train megabase; you cannot visually inspect 1,200+ stations
- The train overview screen (855 trains, 1,231 stations) is the primary debugging tool for identifying supply/demand mismatches
- Missing monitoring circuits on new builds cause false warnings; always copy the monitoring setup from existing builds
- Stone brick shortages were caused by two issues: unbalanced loading (design flaw) and insufficient train count
- Iron warnings persisted despite massive expansion because new stations lacked monitoring circuits, making them invisible to the dashboard

### Design Principles
- Every production station must have monitoring circuits that report to the central dashboard; stations without monitoring are blind spots
- When balancing train loading, ensure input belts distribute evenly to all chests; an unbalanced build causes cascading failures
- Fix root causes in order: (1) check if it is a monitoring/visibility issue, (2) check train count vs station count, (3) check actual production capacity
- Adding trains is the cheapest fix; adding production capacity is the most expensive but most permanent fix
- Stone brick, plastic, and iron are the three resources that need the most headroom above demand

### Ratios & Numbers
- Total network: 855 trains, 1,231 stations
- Stone brick: scaled to 6 trains with 6 stops (was 3 trains/5 stops causing intermittent shortages)
- Train buffer threshold: 32,000 items triggers train request (for standard 4-wagon trains)
- Monitoring signal: each station reports inventory via circuit network to central dashboard
- When demand exceeds supply by more than buffer capacity, the dashboard shows a warning

### Mistakes to Avoid
- Do not forget to add monitoring circuits when building new stations; invisible stations cause phantom shortages
- Do not assume balanced input means balanced output; verify actual chest fill levels
- Do not have fewer trains than stations without accounting for travel time; distant stations will starve
- Copper shortages can be "mind-boggling" but real; do not dismiss persistent warnings as false positives
