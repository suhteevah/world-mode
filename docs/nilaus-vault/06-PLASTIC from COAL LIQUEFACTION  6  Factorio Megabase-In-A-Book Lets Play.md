## Summary
### Key Lessons
- Coal liquefaction is chosen over advanced oil processing because it simplifies the supply chain to just coal in, plastic out (plus water)
- Balancing coal consumption between liquefaction and plastic production is the main design challenge
- Community help is valuable: Jeff S from the community helped refine the liquefaction design after Nilaus spent hours on it
- Console commands can remove resources that interfere with build locations (remove nearby resources command from Factorio wiki)
- This build unlocks the path to red circuits (need copper + green circuits + plastic)

### Design Principles
- Coal liquefaction block: coal inbound via train, plastic outbound via train, water sourced locally
- The design fits in 2 city blocks (stackers + trains in one, production in the other) instead of 3
- Tileable liquefaction design: each refinery consumes 45 coal, and the plastic production also needs coal - must balance both
- Plastic stacks to 100 (same as plates), so the same provider station values work for plastic outbound
- Coal stacks to 50 (same as ore), so the same requester station values work for coal inbound
- Spidertrons ("building buddies") do construction while you focus on design in an adjacent area

### Ratios & Numbers
- Coal liquefaction: each refinery consumes 45 coal per cycle
- Plastic stack size: 100 (same as metal plates, so 32,000 per train)
- Coal stack size: 50 (same as ore, so 16,000 per train)
- Block size: 2 city blocks wide (trains/stackers + production), not 3
- Build progression so far: steel, iron, copper, green circuits, now plastic

### Mistakes to Avoid
- Do not forget that water is needed for liquefaction - plan for offshore pump placement or pipe routing
- Do not consume all coal in liquefaction leaving none for plastic chemical plants
- Do not skip the tileable design step - a non-tileable liquefaction setup cannot scale
- Windows Update can reset microphone settings - always check audio before recording (meta lesson from Nilaus)
