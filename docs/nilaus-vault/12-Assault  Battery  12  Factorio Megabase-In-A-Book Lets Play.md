## Summary
### Key Lessons
- Battery production requires massive sulfuric acid (1,200/sec) - plan fluid logistics carefully
- Building batteries unlocks accumulator production, which is prerequisite for solar panel deployment at scale
- When the megabase outer base is overscaled for current needs, use it for module and solar panel production while slowly building up science
- Modular design pays off: reusing liquid input/output blueprints across different builds saves time
- Gradually retire starter base production as outer megabase takes over (e.g., stop module production in main base to free blue circuits)

### Design Principles
- Build batteries close to sulfuric acid production to minimize fluid transport
- Use modular input/output station blueprints that can be adapted for different products
- When a build needs 3 inbound + 1 outbound, use the 5-connection city block template
- Set train departure condition to "inactivity 2 seconds" for partial unloads rather than "empty" which may never trigger
- Freeing up production in starter base (like modules) redirects blue circuits where needed most

### Ratios & Numbers
- Battery build: ~45 batteries per assembler setup, needs <2 full blue belts input
- Sulfuric acid requirement: 1,200/sec for full battery production
- Battery stack size: 200, so 64,000 per train station target (with 200 stack x 4 wagons adjustments)
- Blue circuit production: 3.5/sec baseline, increased to 4.3/sec after adding more assemblers
- Module train trigger: send when total reaches 8,000 across both types (half-full strategy)

### Mistakes to Avoid
- Do not remove perimeter defenses before replacements are built - a single biter breach caused damage deep inside base
- Do not use blue inserters where green inserters suffice - wastes resources
- Having too many trains on a route without enough supply causes trains to block each other at distant stations
- Forgetting to switch trains from manual to automatic mode after building them
